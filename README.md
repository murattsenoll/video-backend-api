# Video İndirme Backend API

## Kurulum:

### Railway ile Deploy (ÜCRETSİZ):

1. https://railway.app/ 'a git
2. GitHub ile giriş yap
3. "New Project" → "Deploy from GitHub repo"
4. Bu klasörü seç
5. Otomatik deploy olacak
6. URL'yi kopyala (örn: https://yourapp.railway.app)

### Vercel ile Deploy (ÜCRETSİZ):

1. https://vercel.com/ 'a git
2. GitHub ile giriş yap
3. "New Project" → GitHub'dan seç
4. Deploy et
5. URL'yi al

### Local Test:

```bash
npm install
npm start
```

## Kullanım:

API URL'ini Android uygulamasına ekle:
- VideoDownloadService.kt içinde API_BASE_URL değiştir
